Listar los nombres de los empleados cuyo nombre termine con la letra "o".
SELECT nombre FROM Empleado WHERE nombre Like "%o";

Para cada oficio, obtener la suma de los salarios.
SELECT oficio, SUM(salario) FROM Empleado GROUP BY oficio;

Mostrar los datos de los empleados cuyo salario sea mayor que la media de todos los salarios.
SELECT * FROM Empleado WHERE salario >(SELECT AVG(salario) FROM Empleado);

Mostrar el nombre del empleado que tiene el máximo salario.
SELECT nombre FROM Empleado WHERE salario = (SELECT MAX(salario) FROM Empleado);

Mostrar el nombre del empleado que tiene el mínimo salario.
SELECT nombre FROM Empleado WHERE salario = (SELECT MIN(salario) FROM Empleado);
